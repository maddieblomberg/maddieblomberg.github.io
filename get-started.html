MANUAL

*Start assuming that you have already accessed the HPCC, installed code, updated machine file, and compiled the code.

  1. Make a folder to run excitable in $SCRATCH. This will be the run directory.
  2. Put files "les3D-mp", "input.gridx", "input.gridy", and "input.gridz" in the run directory.
  3. Copy the run-time parameter file, "les3D-mp.par" to the run directory
  4. Create folders in run directory called "Restart" and "Snap"
  5. Open "les3D-mp.par" and edit run-time parameters
  6. Run the job using 1 core (non-parallel) $. /les3d-mp
  7. During the run, "logfile" and "history" are created.
    1. You can monitor the run with "logfile" using $ tail logfile in another command window
    2. You can monitor the velocity, turbulent kinetic energy (TKE), and wall-shear stress value (averaged along centerline) using GNUPLOT to read the history file:
      $ gnuplot

Post Processing using Jupyter Notebook

  1. Make sure all snapshots are in one folder
  2. Make sure subfolders follow naming conventions, and create necessary folders to ‘catch’ new data
  3. Execute each cell with code or press play on the top of the notebook
  4. It may take several minutes to process (wait at least 5–10 minutes before suspecting an error)
